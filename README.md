## 스터디 방법

### Step 1) 매주 팀리더가 계획표에 해당하는 내용을 준비해온다. 

계획표를 만들 때, 활용했던 두 권의 책, 책의 설명이 부족하다면 추가적으로 넣는다. 

`Dos and Don'ts` <br>
✓ 흠.. 기본문법에 숫자형 내용이 들어가야겠는걸, 추가해서 준비해온다.      
✘ 문맥설명이 중요도에 대한 이해없이 책에 나온 코드만 똑같이 따라친다.

### Step 2) 스터디한 컨텐츠와 당시 나온 질문들을 잘 정리하여, gitpage에 올린다.
